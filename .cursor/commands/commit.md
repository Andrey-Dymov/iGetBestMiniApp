# Commit + Push

Выполни коммит всех изменений и push в origin.

1. Проверь `git status` — какие файлы изменены
2. Добавь все изменения: `git add -A` (или только изменённые, если есть нежелательные)
3. Создай коммит с кратким сообщением на русском (список изменений без переносов строк)
4. Выполни `git push`

Если пользователь указал сообщение коммита — используй его. Иначе сформулируй по изменениям.
